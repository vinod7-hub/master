JENKINS
Jenkins is an open-source automation tool used to build, test, and deploy software automatically.
It helps developers and DevOps teams automate repetitive tasks in the software development process.
Jenkins is mainly used for CI/CD (Continuous Integration and Continuous Delivery/Deployment).
Continuous Integration (CI)
Developers frequently push code to Git
Jenkins automatically:
Pulls code
Builds it
Runs tests
Continuous Delivery/Deployment (CD)
Jenkins automatically:
Deploys code to servers (Dev/Test/Prod)
Ensures faster releases

Role of Jenkins in DevOps
Jenkins acts as the automation backbone of DevOps.
Key Roles:
Automates CI/CD pipeline
Integrates with tools (Git, Maven, Docker, Kubernetes)
Detects errors early
Improves software quality
Speeds up delivery
Jenkins Plugins
Plugins extend Jenkins functionality.
Examples:
Git Plugin
Maven Plugin
Docker Plugin
Kubernetes Plugin
Slack Plugin
